Información del proyecto:


Hola! Este es un proyecto realizado en un proceso personal de aprendizaje de herramientas de programación creativa en combinación con otras de diseño web. En este caso el objetivo es crear una herramienta con la cual sea posible crear contenido 2D y 3D de forma simple y dentro de un marco de posibilidades otorgado desde el programa.

Invito a que exploren, prueben, generen contenido copado, y lo compartan para que pueda verlo :)

Algunos avisos


¡Si se encuentran con algún error o problema pónganse en contacto! Así puedo tratar de solucionarlo.


Se recomienda usar el navegador chrome, sobre todo para crear videos ya que es el que tiene la mejor compatibilidad hasta el momento.


Al crear videos se guarda un archivo .webm que es necesario convertirlo a mp4. Eso puede hacerse fácilmente en cualquier conversor online, dejo algunas opciones:

El sitio está pensado para ser utilizado primordialmente en un navegador de computadora, pero estoy tratando de habilitar (al menos funciones principales) para mobile y tablets.
